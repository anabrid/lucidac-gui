On using this app

Readme for first time users

Hallo Anabrid-Mitarbeiter! Bitte schaut euch die folgenden Kurzinstruktionen an bevor ihr diese Web-Anwendung durchklickt!

Please note that this application is still in heavy development. You are very much invited to test out what is already possible and report not only problems and bugs, but also expectations or nice-to-have features. Please use one of the following channels to do so:

  • Open a new ticket in the bug tracker of our gitlab project
  • You can also submit a new ticket by mail.
  • Use the Sentry "Report a Bug" button (but read the limitations below)
  • Only as fallback: Make screenshots, write on Mattermost

Please consider the following limitations:

  • Limited mobile support: Please note that this website basically works on mobile phones, but is not (yet) tailored to be used on your smartphone or tablet. That is, it is currently best to be used on a desktop or laptop!
  • Please disable your adblocker for this page, if applicable, in order to allow sentry to work (see below).

On the error tracking software Sentry

To simplify user reports even further, we use the Application Perfromance Monitoring & Error Tracking Software sentry. This allows you to report user feedback even easier by clicking the report a bug button in the lower right, which will automatically also include a screenshot and internal debugging details. A few comments about sentry:

  • We see the issue that sentry silently ignores user feedback. Therefore do not write long texts in the feedback form but use a well tested media instead (such as gitlab ticket).
  • Sentry may be blocked by your adblocker because it can also be used for tracking users (which we don't do). Please consider disabling your adblocker for this page.
  • Sentry is known not to work too well with Firefox (it silently ignores your bug reports). Consider using Chrome or similar browsers (Edge, Safari).
  • Note: We will provide an opt-in/opt-out for sentry performance monitoring in the future to comply with GDPR. Currently the software is in development. We probably also will get rid of sentry in favour of another solution.

This application is open source. The code is hosted at gitlab: lucidac/software/lucidac-gui. You may also consider to have a look at the project README file for further details. The project was initiated by Sven in Feb/March 2024.
