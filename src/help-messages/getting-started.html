Getting started

Ressources and helpful links

LUCIDAC is the world's first discrete fully configurable analog computer made by anabrid. Head over to the LUCIDAC documentation for understanding how to use the device.

Harness the real power of your LUCIDAC by making use of the client libraries for various programming languages as well as the command line interfaces. We offer client libraries for

  • Python, extensive reference implementation: pybrid
  • Minimal Python: shcpy
  • Julia: AnabridREDAC.jl (using REDACProtocol.jl)
  • C and C++: tcredac
  • Typescript/Javascript/ECMAScript: HybridController.ts

Access is possible over various ways:

  • TCP/IP JSON-Lines protocol, also accessible with netcat
  • HTTP JSON-RCP kind-of protocol, also accessible with curl
  • USB-C Serial console speaking the JSON-Lines protocol (Fallback admin interface)

The strongly typed JSON-lines based protocol makes it easy to write a third party custom client in a different programming language.

All LUCIDAC Software is Open Source. You can explore the programming of the Microcontroller unit (MCU) which is called the HybridController.
